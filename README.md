# Product App (Frontend + Backend)

Projeto simples para cadastro de produtos (filmes, músicas, etc.) usando Express + SQLite e um frontend estático.

Estrutura:
- `backend/` - servidor Express e banco `web_03mc.db`
- `public/` - frontend (duas páginas: lista e formulário)

Como rodar:

1. Abra um terminal e vá para a pasta do backend:

```bash
cd "product-app/backend"
```

2. Instale dependências e inicie:

```bash
npm install
npm start
```

3. Abra no navegador:

http://localhost:3000

Banco de dados:
- O arquivo `web_03mc.db` será criado automaticamente em `product-app/backend`.
- A tabela criada é `products_nathan` (substitua se desejar usar outro nome).
